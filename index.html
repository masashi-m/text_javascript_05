1. Vue.js(1)

◆Vue.js とは

・元 Google 社員の Evan You によって制作された JavaScript フレームワーク。
・Web アプリケーションの UI 部分を構築する。
・シンプルな操作を目標に開発された。

● 特徴

・HTML をオブジェクトと見なして操作できる。
・jQuery や JavaScript そのままよりも記述がシンプル。
・双方向データバインディング(後述)の機能を有する。
・PHP や Python、Ruby 等の言語フレームワークと連携が可能


◆ 基本構造

●Vue インスタンス

Vue.js を記述する場合、「Vue インスタンス」を作成します。
(インスタンス：実体の意味。処理・実行されるモノを指す)

Vue インスタンスは Vue.js を記述する際に必須のため、
コードを書いて慣れて行きましょう。
基本構造は以下の形になります。

[JavaScript]

let app = new Vue({
  //実行処理(オプション)
});

●Vue.js の準備

Vue.js は HTML に対して実行しますので、HTML ファイルと
JavaScript ファイルを作成して準備しましょう。

(1)Visual Studio Code を立ち上げる。

(2)デスクトップに「Vue」フォルダ作成

(3)File > Open Folder > Vue を選択して開く

(4)EXPLORER 配下の Vue をクリックし、次のフォルダを作成
・html
・css
・js

(5)作成した各フォルダ配下に次のファイルを作成
・html フォルダ > index.html
・css フォルダ > style.css
・js フォルダ > vue.js

2.Vue.js(2)

●HTML ファイルの記述

引き続き、HTML ファイルの準備を進めて行きます。
さっそく、次の演習を実施しましょう。


演習 1
以下のコードを記述し、Google Chromeを更新して結果を確認しましょう。

[html]

<body>
  <div id="app1">
    <p>
      {{ message }}
    </p>
  </div>
  {{ message }}
</body>

●Mustache(マスタッシュ)構文

上記の演習では、HTML ファイルに対して
(1) div タグで id="app1" の属性を指定、
(2) p タグの中に {{ message }} を記述しました。

これは、この後記述する js ファイルの指定先を作成しています。
つまり、vue.js は id="app1" と {{ message }} を参照する事になります。

二重波括弧の事を 「Mastache 構文」といいます。
Vue.js の記述方式で、JavaScript の式を利用する事が可能です。
また、Mastache 構文内の式を「バインディング(式)」とも言います。


●JavaScript ファイルの記述

次に js ファイルの記述を進めて行きます。
次の演習を実施しましょう。


演習 2
以下のコードを記述し、Google Chromeを更新して結果を確認しましょう。

[JavaScript]

let app = new Vue({
  el: "#app1",
  data: {
    message: "Hello Vue.js!",
  },
});

ブラウザには、js ファイルに記述された 「Hello Vue.js!」の
上で表示され、その下に {{ message }} の文字列が
表示されているはずです。
これらについて、1 つずつ解説して行きましょう。


● データバインディング

js ファイルの Vue インスタンスの中には、2 種類の記述があります。

1 つ目が el(エレメント)です。ここでは、HTML ファイル内の
id:app1 の要素を指定しています。

2 つ目が データオプションです。message に対して Hello Vue.js! の
データを与えています。

Vue インスタンス内でこれらを記述した事により、
「id 属性 app1」の HTML 要素に対し、
「message: 'Hello Vue.js!'」が
連結(データバインディング) された結果、
ブラウザに表示されます。

また、div の外にある {{ message }} がそのままになるのは、
Vue の対象範囲外であり、html として表示されるためです。



● オブジェクトの表示

Mastache 構文には、文字列以外にも数値や式、オブジェクトを
使用する事ができます。次の演習に取り組みましょう。

演習 2
以下のコードを記述し、Google Chromeを更新して結果を確認しましょう。

[html]

<div id="app2">
  <ul>
    <li v-for="todo in todos">{{ todo }}</li>
  </ul>
</div>

[JavaScript]

let app = new Vue({
  el: "#app2",
  data: {
    todos: ["tomato", "carot", "onion"],
  },
});
演習 3
下記のコードを記述し、実行結果を確認しましょう。

[html]

<div id="app3">
  <p>現在時刻は {{ now_time }}</p>
</div>

let now = new Date();

  el: "#app3",
  data: {
    now_time: now.toLocaleTimeString(),
  },
});

4. Vue.js(4)

◆TODO リストの作成

総合演習として「TODO リスト」を作成し、
アプリケーション構築の基礎を学んで行きましょう。


●TODO リストに必要な機能

・テキスト入力フォーム
・追加ボタン
・タスクリストの表示
・チェックボックスと取消し線の表示
・削除ボタンとオブジェクトの削除

以下の演習を進め、html, CSS, JavaScript のコーディングを実施しましょう。


総合演習1-1
デスクトップに TODOフォルダを作成し、JS_5-1と同様に
3つのフォルダを作成します。その後、todo.htmlファイルを
作成し、以下のコードを記述しましょう。

[html]

<!DOCTYPE html>
<html lang="ja">
  <head>
    <meta charset="UTF-8" />
    <title>Application</title>
    <link rel="stylesheet" href="../css/style.css" />
  </head>
  <body>
    <div id="app">
      <h1>ToDo リスト</h1>
      <form v-on:submit.prevent>
        <input type="text" v-model="newItem" />
        <button v-on:click="addItem">追加</button>
      </form>
      <ul v-cloak>
        <li v-for="(todo, index) in todos">
          <input type="checkbox" v-model="todo.isDone" />
          <span v-bind:class="{ done: todo.isDone }">{{ todo.item }}</span>
          <button v-on:click="deleteItem(index)">削除</button>
        </li>
      </ul>
    </div>
    <script src="https://cdn.jsdelivr.net/npm/vue@2.6.10/dist/vue.js"></script>
    <script src="../js/vue.js"></script>
  </body>
</html>

総合演習1-3
jsフォルダに veu.jsファイルを作成後、以下のコードを記述しましょう。
//インスタンスの作成
let app = new Vue({
  el: "#app",
  data: {
    //タスクを格納する変数newItemを定義
    newItem: "",
    //変数todosに配列を定義
    todos: [],
  },

  //methodsオプションの定義
  methods: {
    addItem: function (event) {
      //テキストボックスが空の場合は処理を止める
      if (this.newItem == "") return;

      //変数todoにタスクを格納
      let todo = {
        item: this.newItem,
        isDone: false, //格納時isDoneオブジェクトの値はfalse
      };

      this.todos.push(todo); //配列todosにtodoのオブジェクト：プロパティを格納
      this.newItem = ""; //テキストボックスの文字をクリア
    },
    //deleteItemメソッドの定義
    deleteItem: function (index) {
      this.todos.splice(index, 1); //リストを削除
    },
  },
});
